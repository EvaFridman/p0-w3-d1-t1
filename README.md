## Задача

Первый день JS — учишься различать, какие бывают значения и как спросить у языка их тип через `typeof`. Заводишь файл, складываешь по значению каждого примитива и печатаешь их тип.

```
значение  →  typeof
  "hi"     →  "string"
  42       →  "number"
  true     →  "boolean"
  undefined→  "undefined"
  null     →  "object"  ← историческая засада
  10n      →  "bigint"
```

## Что нужно сделать

Создать `types.js`. Объявить переменные со значениями каждого примитива (string, number, boolean, undefined, null, bigint, symbol) и вывести `typeof` каждого в консоль. Рядом коротким комментарием — что получилось.

## Критерии приёмки

* Выведен `typeof` для всех семи примитивов.
* В комментарии отмечено, что `typeof null` возвращает `"object"` — известная историческая особенность, а не твоя ошибка.
* Можешь объяснить, чем `null` отличается от `undefined` по смыслу.
